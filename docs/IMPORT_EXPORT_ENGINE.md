# Moteurs d'import et d'export

Statut: en attente du checkpoint Phase 5 import/export.

L'import CSV de contacts historique reste disponible dans le module connecteurs. Il ne constitue pas encore le moteur universel demandé.

La cible Phase 5 doit ajouter prévisualisation, mapping, validation, dry-run, dédoublonnage, traitement borné, reprise, rapport d'erreur et retour arrière pour CSV, XLSX et JSON. Les exports devront être tenant-scoped, limiter les champs sensibles et les plages, expirer et être audités.
