# Website Factory

Le site n'est pas genere comme code arbitraire. Il est compose de donnees :

- `websites`
- `website_pages`
- `website_sections`
- `website_versions`
- `website_publications`
- theme, SEO metadata, formulaires, CTA

Sections MVP : hero, introduction, services, benefits, FAQ, horaires, zone, contact, footer.

Templates : artisan/services, restaurant/hospitalite, beaute/rendez-vous.

L'editeur permet activation, ordre, titres, textes, images, boutons, apercu desktop/mobile, sauvegarde, publication et restauration de version.
